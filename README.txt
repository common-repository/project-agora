Easily implement the Project Agora header tag code on your site, without the need of a developer.

== Description ==

This plugin enables WordPress users to easily implement the Project Agora header tag code on their site, without the need of a developer.
Once you install the plugin, use the code you received from Project Agora during onboarding.
An automatic email will be sent to Project Agora notifying that you are ready to start monetizing your site.
